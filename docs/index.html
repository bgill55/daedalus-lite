What's under the hood

~300 lines of TypeScript. Middleware chain. Multi-model router. No magic.

router.ts
// Route to OpenAI, Anthropic, or local models
async dispatch(request: ModelRequest): Promise { const model = request.model || this.config.defaultModel; if (model.startsWith('gpt-') || model.startsWith('o1-')) { return this.dispatchToOpenAI({ model, messages }); } else if (model.startsWith('claude-')) { return this.dispatchToAnthropic({ model, messages }); } else { return this.dispatchToLocal({ model, messages }); }
}
middleware.ts
// Middleware chain - add logging, auth, rate limiting
use(middleware: Middleware): void { this.middleware.push(middleware);
} // Error handling middleware
export const errorHandlerMiddleware = async ( req: ModelRequest, next: () => Promise
): Promise => { try { return await next(); } catch (error) { return { choices: [{ message: { content: "Error: " + error.message } }] }; }
};

What You Can Build with Daedalus-Lite

Daedalus-Lite's 3-step middleware architecture lets you intercept, enrich, and format AI interactions for any domain. Here is what's possible:

Financial & Compliance Assistant

Prepend portfolio context and automatically enforce regulatory disclaimer injection on output.

export const financialGuard: Middleware = async (req, next) => {
  req.prompt = `Add financial context: ${req.prompt}`;
  const res = await next();
  res.content += '\n Not financial advice.';
  return res;
};

Legal Document Analyzer

Parse contract clauses and enforce strict JSON schema responses with automated validation.

export const legalJsonMiddleware: Middleware = async (req, next) => {
  req.prompt = `Respond ONLY in JSON: ${req.prompt}`;
  const res = await next();
  JSON.parse(res.content); // validate schema
  return res;
};

Automated Code Review Bot

Scan git diffs for security vulnerabilities, memory leaks, and generate structured PR checklists.

export const codeReviewer: Middleware = async (req, next) => {
  req.prompt = `Review for security & style:\n${req.prompt}`;
  return next();
};

Medical Triage Assistant

Categorize patient symptoms into urgency levels (Critical, Urgent, Routine).

export const medicalTriage: Middleware = async (req, next) => {
  req.prompt = `Classify symptom urgency:\n${req.prompt}`;
  return next();
};

Customer Support RAG Agent

Intercept customer tickets, inject relevant knowledge base articles, and log sentiment.

export const supportRAG: Middleware = async (req, next) => {
  const kb = await searchDocs(req.prompt);
